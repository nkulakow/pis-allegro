# pis-allegro
System działający na podobnej zasadzie jak Allegro.

## Wybrane technologie:
- **język**: Java + Spring
- **automatyzacja budowy projektu**: Maven
- **repozytorium kodu**: GitHub
- **repozytorium mavenowe**: Nexus
- **serwer CI**: Jenkins
- **IDE**: Intellij IDEA
- **issues**: Jira
- **testy**: JUnit
- **narzędzie do mierzenia pokrycia kodu testami**: JaCoCo
